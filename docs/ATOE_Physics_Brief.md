# ATOE Physics Brief (Open-Source Core)

**Abhishek’s Theory of Everything (ATOE) – Open Core Documentation**  
Version: 1.0 (April 2026)

This document explains **only** the physics and mathematics that are included in the open-source `MasterSLWSolver.py`.

## What is Open-Sourced

The open-source code implements the following core ATOE concepts:

1. **Coherence Scoring**  
   Measures how well two frequencies phase-lock using harmonic ratio and a simple causal factor.

2. **Phase-Band Stability Check**  
   Implements the fundamental ATOE condition:  
   Δϕ_stable × Δr ≥ π

3. **Standing-Wave / Vastu Resonance Scoring**  
   Calculates resonance quality for cylinders, rooms, temples, etc., using golden-ratio geometry and wavelength fit.

4. **Material Science Optimiser**  
   Suggests better materials (copper_graphite, silver_graphite, graphene_coated, etc.) to improve SLW coherence and Q-factor.

5. **Basic Resonator Optimisation**  
   Combines the above tools to optimise simple resonator geometries (e.g., golden-ratio cylinder).
